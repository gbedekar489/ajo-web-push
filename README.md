# AJO Web Push Notifications Demo

This project demonstrates how to implement **web push notifications using Adobe Journey Optimizer (AJO)** and **Adobe Experience Platform (AEP)**.

It covers the full flow:
- Capturing user opt-in using the Web SDK
- Registering push subscriptions in AEP
- Sending push notifications via campaigns
- Triggering real-time journeys using a custom `price.drop` event

## 📌 Features

- Web push opt-in using the **Notifications API**
- Push subscription registration via **AEP Web SDK**
- Campaign-based push messaging in AJO
- Event-driven journeys using a custom `price.drop` event
- Multi-product event triggering using Adobe Data Layer
- Node.js backend for dynamic configuration (`/config` endpoint)
